# 我用 Claude Code 6 天做了产品，过程中发现一个比产品更值得做的事

> 不是"AI 帮人做事"。是"人帮 AI 做事"——一个新品类。

---

我用 Claude Code 6 天做出了一个产品。但中间我一直在干一件事：调 prompt、看输出、切模型、换参数、记日报。所有这些操作没有一个统一的地方管理。我在命令行和浏览器之间来回切。

后来我发现，这个问题不是我一个人的。

**每一个拿 Claude Code 干活的人，都在"操作 AI"上花的时间超过了"自己写代码"的时间。但所有工具——IDE、终端、浏览器——都是为写代码的人设计的。没有一个是为操作 AI 的人设计的。**

这就是 Human-for-AI。

---

## 你的 Claude 在烧钱，你知道吗？

说几个场景，看有没有戳中你：

**场景一：月底账单来了，一脸懵。**

你打开 DeepSeek 后台，¥2,000。花哪了？哪个项目最烧钱？那次调 prompt 反复试了 20 轮，烧了多少？不知道。你只知道总额。

**场景二：你怀疑你的渠道在宰你。**

朋友说他用的渠道 ¥1.0/百万 token，你的 ¥3.0。是模型不一样？还是你被坑了？你懒得查，因为查起来太麻烦。

**场景三：今天聊嗨了，不知道花了多少。**

跟 Claude 来回调了一个下午，改 prompt→重跑→再改→再重跑。退出的时候看了一眼：¥380。心态崩了。"我 tm 在干嘛？"

**场景四：老板问 AI 成本，你答不上来。**

"咱们这个月 AI 花了多少？""哪个项目投入产出比最高？""能不能给个报告？"——你只能打开后台手动翻。

---

## 我问了 Claude："我今天花了多少？"

这是 Claude 的回答：

```
让我读取你的 session 文件...
grep jsonl... 解析 usage... 手算 token...
(消耗 2000 token 做数据查询)
(不知道模型价格，只能告诉你 token 数)
(没有历史对比，没有建议)
```

花了 2000 token 去查账，结果只得到一个数字。而且不直观。

**这就是裸 Claude 查账的体验——能做，但做不好。因为查账不是它擅长的。**

## 然后我装了一个 Skill

装完以后，同一句话，Claude 跑了一行命令：

```
/cc-cost

📊 CC-COST  2026年6月3日 周三
────────────────────────────────────────
  今日消耗  ¥5.51
  缓存命中 159.8M tokens  ·  省了 ¥156.57

  项目              输入     输出      费用
  🎭 剧本宇宙         75K     90K   ¥0.76
  📝 博客            42K     18K   ¥0.68

  本周  06/01 → 06/08  ¥127.50
  较上周 -15% ↓  ·  34次会话  ·  缓存命中 92%

  状态
  🏦 比价: ⚠️ 你的渠道比官方贵 100%
  💡 建议: V4 Pro → V3 可月省 ¥40
  💳 预算: ¥234/¥500 (47%)
────────────────────────────────────────
```

**一张仪表盘。今日+本周+比价+省钱建议+预算预警。一屏全出来。**

而且 Claude 看完数据以后，自己做了三件事：

1. **解读**："剧本宇宙占 86%，6/1 那天异常飙升，大概率在循环调试"
2. **调行为**："今天简单任务用 V3，大项目先写 CLAUDE.md，少开新 session"
3. **记教训**：自动把成本发现写入 memory，下次对话还记得

**不是"你盯着仪表盘省钱"。是"Claude 看了仪表盘，自己省钱"。**

## 为什么是 Human-for-AI

现在所有人都在做"AI for human"——让 AI 帮人写代码、写文章、做设计。这是第一波。

第二波是反过来的——**Human for AI**：人帮 AI 更好地工作。

| | AI for Human | Human for AI |
|---|---|---|
| 例子 | ChatGPT、Copilot、豆包 | cc-cost、prompt 管理、agent 运维 |
| 谁在操作 | AI 替人干活 | 人替 AI 管账、调参、优化 |
| 用户心态 | "免费也能用 80%" | "这是在省真金白银" |
| 竞品 | 满大街都是 | 还没人做 |
| 付费意愿 | 低（总是跟免费比） | 高（在做 ROI 计算，不是比价） |

**AI 操作者（AI Operator）是一个正在成型的新职业。** 现在连名字都没有，但三年后每个公司都会有。这个职业需要的工具链，现在几乎一片空白。cc-cost 是第一个——Agent 的记账本。

## 怎么装

在 Claude Code 里粘贴这句话：

```
请帮我安装这个插件：https://github.com/Hedyzhang-Ading/cc-cost
运行 install.sh 即可。
```

或者在终端：

```bash
curl -fsSL https://raw.githubusercontent.com/Hedyzhang-Ading/cc-cost/main/install.sh | bash
```

然后敲 `/cc-cost`。一屏全出来。

**开源。零依赖。零网络请求。你的数据只在你本机。**

GitHub: https://github.com/Hedyzhang-Ading/cc-cost
